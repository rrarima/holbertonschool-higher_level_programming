The ``MyList`` module
======================

Using ``MyList``
-------------------


Import ``my_list``:

     >>> MyList = __import__('1-my_list').MyList

Now use it:

    Basic sort:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list.print_sorted())
    [1, 2, 3, 4, 5]
    None

    Sort negative numbers:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list.print_sorted())
    [-1, 2, 3, 4, 5]
    None

    Sort empty list:
    >>> my_list = MyList()
    >>> print(my_list.print_sorted())
    []
    None

    Sort using __str__:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list.__str__())
    [1, 4, 2, 3, 5]

    Sort return new list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> print(my_list.print_sorted())
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]
