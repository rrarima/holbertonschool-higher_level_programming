The ``MyList`` module
======================

Using ``MyList``
-------------------


Import ``my_list``:

     >>> MyList = __import__('1-my_list').MyList

Now use it:

    Basic sort:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list.print_sorted())
    [1, 2, 3, 4, 5]
    None
